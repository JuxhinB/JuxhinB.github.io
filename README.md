## üõ†Ô∏è Setup & Deployment

### Prerequisites

- A GitHub account
- This repository named `JuxhinB.github.io` (or your username)

### Deployment Steps

1. **Enable GitHub Pages**
   - Go to your repository settings
   - Navigate to "Pages" in the left sidebar
   - Under "Source", select the `main` branch
   - Click "Save"

2. **Your site will be live at:**
   ```
   https://juxhinb.github.io
   ```

3. **Custom Domain (Optional)**
   - Add a `CNAME` file with your domain name
   - Configure DNS settings with your domain provider

### Update Personal Information

1. **Experience Section** (`index.html`)
   - Edit the `.experience-item` divs with your work history
   - Update dates, titles, companies, and descriptions

2. **Skills Section** (`index.html`)
   - Modify the `.skill-tag` elements with your actual skills
   - Add or remove skill categories as needed

3. **Education Section** (`index.html`)
   - Update the `.education-item` with your academic background

4. **About Section** (`index.html`)
   - Customize the about text to reflect your personal story

- Edit `styles.css` to change colors, fonts, spacing, etc.
- Color scheme is defined in CSS variables at the top of `styles.css`

- Modify `script.js` to add or change interactive features
